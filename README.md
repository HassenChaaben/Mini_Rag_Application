# mini-rag 

This is minimal implementation of the RAG model for question answering.

## Requirements 

- Python 3.8 or later 

### Install Python using MiniConda 

### Install python using MiniConda 

1) Download and install MiniConda From [here] (https://www.anaconda.com/docs/getting-started/miniconda/install)

2) Create a new environnement using the following command:
```bash
$ conda create -n mini-rag python=3.8
```
3) Activate the environment:
 ```bash
$ conda activate  mini-rag-app
```
### (Optional) Setup you command line interface for better readability

```bash
export PS1="\[\033[01;32m\]\u@\h:\w\n\[\033[00m\]\$ "
```

## Installation

### Install the required packages

```bash
$ pip install -r requirements.txt
```
